# OAKUI
> A Wechat App UI Lib.
<br/>

## Development

### get start
```code
$ git clone git@github.com:yh-yunchuang-fe/oak.git
$ cd oak
$ npm install
$ npm start
```

### build
```code
npm run build
```

### new
```code
npm run new `ComponentName`
```
