# ListItem

## 使用

在页面 `json` 中引入按钮组件：

```json
// import in `page.json`:
"usingComponents": {
  "oak-list-item": "path/to/your/oakui/ListItem/list-item"
}
```

在页面使用
```html
<!-- use in `page.wxml` -->
<oak-list-item />
```
