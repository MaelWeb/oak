# Button 按钮

### Usage

Here is a quick example to get you started, it's all you need:

import in `page.json`:
```json
"usingComponents": {
  "oak-button": "path/to/your/oakui/button/index"
}
```
use in `page.wxml`
```html
<oak-button isShow />
```
